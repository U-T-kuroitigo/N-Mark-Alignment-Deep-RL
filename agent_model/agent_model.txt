
agentは以下のように保存される

モデルディレクトリ = モデルタイプ_盤面_N目並べ_何人対戦
バージョン = モデルディレクトリ_学習回数
バージョンディレクトリ = バージョン_タイムスタンプ
モデルディレクトリ内に
📦モデルディレクトリ
 ┣ 📂バージョンディレクトリ
 ┣ 📂バージョンディレクトリ
 ┗ 📜evaluation_history.csv     # モデル学習履歴

バージョンディレクトリ内に
📦バージョンディレクトリ
 ┣ 📂meta
 ┃ ┣ 📜バージョン_タイムスタンプ.json   # メタデータ1
 ┃ ┗ 📜バージョン_タイムスタンプ.json   # メタデータ2
 ┣ 📂replay
 ┃ ┣ 📜バージョン_タイムスタンプ.pkl    # リプレイバッファ1
 ┃ ┗ 📜バージョン_タイムスタンプ.pkl    # リプレイバッファ2
 ┗ 📜バージョン_タイムスタンプ.pt       # モデルの重み
のように保存される
